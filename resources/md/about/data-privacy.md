# Data Privacy Policy

## Overview

The uPortal project is open-source software developed under the [Apereo Foundation](https://www.apereo.org/). This policy covers the uPortal project website and community infrastructure, not individual institutional deployments of uPortal (which are governed by each institution's own privacy policies).

## Information We Collect

**We do not collect personal information.** The uPortal project website is a static site that does not use cookies, analytics trackers, or any form of user data collection.

## Information We Do Not Sell

We do not sell, trade, or otherwise transfer any information to third parties. Even in the event that we were to collect any information in the future, we would never sell it.

## Mailing Lists and Community Participation

When you subscribe to a uPortal mailing list or participate in community channels, you voluntarily provide your email address and any content you post. Mailing list archives are public. This is standard for open-source projects and is governed by the [Apereo Foundation's privacy practices](https://www.apereo.org/).

## Code Contributions

When you contribute code to uPortal repositories, your name and email address are recorded in the Git commit history. This is inherent to how Git works and is standard practice for open-source development.

## Institutional Deployments

uPortal is a portal framework that institutions deploy and configure independently. **The uPortal project has no access to any data within institutional deployments.** Each institution that deploys uPortal is responsible for their own data privacy policies, data handling practices, and compliance with applicable regulations (such as FERPA, GDPR, or other local laws).

The uPortal software itself is designed to respect user privacy and provides configuration options for institutions to manage data retention, access controls, and user consent in accordance with their own policies.

## Third-Party Services

This website may link to external services such as GitHub, mailing list archives, and the Apereo Foundation website. These services have their own privacy policies, and we encourage you to review them.

## Changes to This Policy

Any changes to this policy will be reflected on this page. As an open-source project, significant policy changes would be discussed on the community mailing lists.

## Contact

Questions about this policy can be directed to the [uPortal community mailing lists](../community/mailing-lists.html) or the [Apereo Foundation](https://www.apereo.org/).
